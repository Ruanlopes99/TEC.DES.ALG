# Avaliação A1 - Técnicas de Desenvolvimento de Algoritmos

Este projeto reúne os 4 exercícios práticos em um único sistema com menu - Avaliação A1.

## Como executar
Basta rodar o arquivo `projeto_completo.py`. Um menu aparecerá na tela para você escolher qual exercício testar:

1.  **Verificar Idade:** Testa condicionais (if/elif/else).
2.  **Contar Pares:** Testa repetição (for e while) de 1 a 100.
3.  **Lista de Alunos:** Testa inclusão e leitura de listas.
4.  **Cadastro de Produtos:** Testa dicionários com nome e preço.
